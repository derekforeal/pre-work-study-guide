Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • UL and OL create unordered and ordered lists. To create list items we use li.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • Padding indicates how much space we want around the content inside of an element.
  • An * applies a rule to all elements visible on the page.
  • A period is a class selector. Any HTML element assigned to this class will gain the CSS properties.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: returns us to main branch
